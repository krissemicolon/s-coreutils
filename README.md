## Installation  
1. Clone Repository  
```
git clone https://github.com/krissemicolon/s-coreutils
```
2. Cd into Repository    
```
cd s-coreutils
```
3. Compile  
```
make
```
4. Moving Binaries to /bin  
```
make install
```

## Uninstall  
To uninstall you just need to run  
```
make uninstall
```
in the c-coreutils repository.
